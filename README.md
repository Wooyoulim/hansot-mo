
# UIUX 웹앱 디자인 & 프론트엔드 개발 국비지원 수업 팀 프로젝트

한솥홈페이지 리디자인 (모바일편)

## 프로젝트 소개
한솥 메뉴들로 만날 수 있는 꿀조합 레시피 페이지를 추가하여 새롭게 디자인한 리뉴얼 모바일 홈페이지
<br>
## 개발 기간
*24.4.29 ~ 24.5.3

### 멤버구성

- 이탐라 - 모바일 페이지 전체디자인, 메인페이지 및 가맹점 소개
- 우유림 - 헤더 및 푸터, 메뉴바, 로그인, 자주 묻는 질문
- 박해윤 - 브랜드 스토리, 메뉴, 고객의 소리
- 이의강 - 꿀조합 레시피, 점포찾기, 지도 API, 발표
